# Folha de Runeterra

Projeto desenvolvido para praticar HTML semântico, organização de layout e conceitos iniciais de responsividade.

## Tecnologias

- HTML5
- CSS3

## Objetivo

Este projeto foi criado com o objetivo de treinar:

- HTML semântico
- Estruturação de blogs e páginas de conteúdo
- Organização de seções e componentes
- Responsividade básica
- Estilização com CSS
- Boas práticas de desenvolvimento web

## Funcionalidades

- Estrutura de blog
- Layout responsivo inicial
- Seções organizadas semanticamente
- Uso de imagens e backgrounds personalizados

## Estrutura do Projeto

```bash
📦 folha-de-runeterra
 ┣ 📜 index.html
 ┣ 📂 styles
 ┃ ┗ 📜 style.css
 ┣ 📂 assets
 ┃ ┣ 📜 background.jpg
 ┃ ┣ 📜 background2.jpg
 ┃ ┣ 📜 img1.webp
 ┃ ┣ 📜 img2.webp
 ┃ ┗ 📜 img3.webp
 ┗ 📜 README.md
```

## Status do Projeto

Projeto finalizado para fins de estudo e prática.
